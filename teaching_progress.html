將課表照片給 Claude 解析後，把 Claude 產生的 JSON 貼在下方匯入。格式範例：
[{"name":"313","schedule":[{"period":"第2節","isFlex":false,"hasEighth":false},null,null,{"period":"第1節","isFlex":true,"hasEighth":false},{"period":"第4節","isFlex":false,"hasEighth":true}]}]
schedule 陣列共5個元素，依序為週一到週五，無課填 null。

將行事曆 PDF 給 Claude 解析後，把 Claude 產生的 JSON 貼在下方匯入。格式範例：

type 可填：exam（段考）、review（複習考）、holiday（放假日）、other（其他）
