### 2026-06-29｜classroom_record.html / classroom_record_5class.html 功能升級

#### 本次完成

**`classroom_record_5class.html`（五班版）新增功能**

- 班級名單匯入：支援 Excel（.xlsx）與 CSV 格式，第一欄座號、第二欄姓名，匯入後座號格自動顯示姓名；切換班級時自動更新對應名單
- 特殊班級記事：自動帶入上方已選班級與日期，每班每日一則，支援儲存、查看（切換顯示/隱藏）、編輯、刪除；記事會一併納入匯出檔案
- 匯出格式升級：改為 Excel（.xlsx）多分頁格式，每個班級各一個工作表
  - 「期末統整」（統整頁）：含座號、姓名、基本分、加扣分、課堂分數、記錄次數、功課未交、上課講話，及下方特殊班級記事
  - 「明細記錄」（設定頁）：含日期、座號、姓名、項目、分數，及下方特殊班級記事
  - 檔名格式：`課堂紀錄期末統整_日期.xlsx`、`課堂紀錄明細記錄_日期.xlsx`
- SheetJS 函式庫：已內嵌於 HTML 檔案，離線可用

**`classroom_record.html`（四班版）同步更新**
- 與五班版同步所有上述新功能（名單匯入、記事、Excel 匯出）
- 儲存鍵值與五班版分開（`cr_records` / `cr_config` / `cr_notes_v1` / `cr_roster_v1`），資料不互相影響

#### 修改的檔案

### 2026-06-28｜語音成績登記系統（全新工具）

**修改人：** Mia（搭配 Claude 協助）

#### 本次完成

**新增 `grade_voice_system.html`（v1.5）**，完整開發歷程如下：

**v1.0 初版**
- 五個班級架構，名稱可自訂，成績項目可設定並支援跨班複製
- 班級名單匯入（格式：`座號,姓名`）
- 語音辨識核心：辨識「N號M分」自動填入對應座號
- 支援阿拉伯數字與中文數字混合輸入
- 統計分析：平均分、各分段人數（90↑／80–89／70–79／60–69／60↓）與分布圖
- 單班 CSV 匯出；Safari / iPad 語音辨識相容設計

**v1.1 語音辨識強化**
- 新增 localStorage 自動儲存，重新整理後資料不遺失
- 修正中文數字解析：支援「兩字裸數字」（六一→61、七八→78）
- 新增諧音詞替換表（爸爸→88、酒吧→68 等 Safari 常見誤辨）
- 修正單字數字判讀：「八」無「分」字→88；有「分」字→8
- 新增「最近填入」標籤，點擊可直接修正辨識錯誤

**v1.2 照片辨識匯入**
- 新增照片辨識匯入（Claude JSON 貼入方案，完全免費）
- 流程：複製提示詞 → 連同照片發給 Claude → 貼回 JSON → 預覽確認 → 匯入
- 多科目一次匯入，新科目自動建立，無需預先設定

**v1.3 重大 Bug 修正**
- 修正 `panel-settings` 缺少結尾 `</div>`，導致其他三個 tab 全部空白
- 修正 Safari `event.currentTarget` 問題（Safari 不支援全域 event），改為傳入 `this`
- 移除殘留的 `updateApiKeyStatus()` 呼叫，修復頁面初始化中斷
- 移除重複的 `selectGradeClass` 定義，消除 JS 衝突

**v1.4 語音解析精化**
- 修正停頓後 Safari 重新辨識整段導致舊成績被覆蓋的問題
  - 解法：`onresult` 只取 `event.resultIndex` 之後的新片段
- 分數正則限制為 0–100 有效範圍，防止誤辨數字被當成座號
- 加分題超過 100 分自動截為 100

**v1.5 匯出升級**
- 匯出時自動略過空白座位（無姓名且無成績的座號）
- 全班匯出改為單一 `.xlsx` 檔案，各班各一個工作表分頁（使用 SheetJS）

**同步更新 `index.html`**
- 語音成績登記系統說明欄新增「語音速度建議」與「即時確認分數」提示
- 功能標籤補上：照片辨識匯入、統計分析、匯出 Excel

#### 修改的檔案
- `grade_voice_system.html`（新增）
- `index.html`（更新說明欄）
- `PROGRESS.md`（本次更新）

#### 做了哪些決策
- 照片辨識放棄 Tesseract.js（Safari 相容性差）與 Claude Vision API（需額外付費），改採「提示詞＋Claude 對話辨識＋JSON 貼回」方案，免費且準確率最高
- 全班匯出從多個 CSV 改為單一 xlsx，減少操作步驟
- 語音辨識採 `continuous:false` + 自動重啟，規避 Safari 不支援 continuous 模式的限制

#### 目前卡點 / 待確認
- 語音辨識仍受 Safari 語意校正影響，諧音詞替換表無法涵蓋所有情況
- 建議每次使用後確認「最近填入」欄位，有誤點擊修正

#### 下次要做
- 視使用情況決定是否補充更多諧音詞
